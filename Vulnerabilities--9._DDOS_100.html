9. DDOS


A Distributed Denial of Service (DDoS) attack is a type of cyber attack where multiple compromised systems, often called "botnets," are used to flood a targeted server or network with traffic in order to overwhelm it and cause it to become unavailable.

In web application security, DDoS attacks can be used to take down websites or web applications by overwhelming their servers with traffic. This can result in slow response times, website downtime, or even complete server failure.
