3. Content Security Policy (CSP)


  • Is a W3C specifcation ofering the possibility to instruct the client browser from which location and/or whichtype of resources are allowed to be loaded. To defne a loading behavior, the CSP specifcation use “directive”where a directive defnes a loading behavior for a target resource type. CSP helps to detect and mitigate certain types of attacks, including Cross Site Scripting (XSS) and data injection attacks. These attacks are used for everything from data theft to site defacement or distribution of malware
